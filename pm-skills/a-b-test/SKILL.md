# A/B Test - A/B测试设计

---

## Description

科学设计 A/B 测试，验证产品改动的真实效果。

**解决的实际问题：**
- 凭感觉上线，不知道效果好不好
- 测试设计不严谨，结果不可信
- 样本不够就下结论，误判

---

## When to Use

**强制触发条件（必须同时满足）：**
1. 有明确要验证的改动（按钮颜色/文案/流程等）
2. 有明确的指标提升目标
3. 有充足的样本量（或能计算所需样本）

**不适用场景：**
- 样本量太小（<1000人/组）
- 改动涉及合规/安全风险（需全量谨慎上线）
- 临时性/一次性活动（无法持续观察）

---

## Pre-conditions

**必须提供：**
- 改动描述（A版本 vs B版本）
- 核心指标（如点击率、转化率）
- 当前基线数据（如当前转化率5%）

**可选提供：**
- 预期提升幅度
- 可接受的风险水平

---

## Workflow

### Phase 1: 假设定义

```markdown
## 测试假设
- 原假设 H0：新方案与旧方案无差异
- 备择假设 H1：新方案转化率 > 旧方案

## 预期效果
- 当前转化率：5.0%
- 预期提升到：6.0%（+20%相对提升）
- 最小可检测提升：1个百分点
```

### Phase 2: 样本量计算

```python
# 简化公式（95%置信度，80%功效）
n = 16 * σ² / δ²

# 实际计算
from statsmodels.stats.power import NormalIndPower
from statsmodels.stats.proportion import proportion_effectsize

# 参数
p1 = 0.05  # 对照组转化率
p2 = 0.06  # 实验组预期转化率
alpha = 0.05  # 显著性水平
power = 0.8   # 统计功效

# 计算所需样本量
effect_size = proportion_effectsize(p1, p2)
power_analysis = NormalIndPower()
sample_size = power_analysis.solve_power(
    effect_size=effect_size,
    alpha=alpha,
    power=power,
    ratio=1
)

print(f"每组需要样本量: {int(sample_size)}")  # ≈ 12,000
print(f"总样本量: {int(sample_size * 2)}")     # ≈ 24,000
```

**样本量结果：**
```markdown
## 样本量要求
- 每组最少：12,000人
- 总样本量：24,000人
- 预期用时：当前日活10万，分流10%，约5天
```

### Phase 3: 测试设计

```markdown
## 分组策略
- 分组方式：用户ID哈希取模（保证同用户始终同组）
- 分流比例：A组50% : B组50%
- 白名单：内部测试账号固定到B组

## 观察指标

### 核心指标（决策依据）
| 指标 | 定义 | 基线值 | 目标值 |
|------|------|--------|--------|
| 转化率 | 下单UV/访问UV | 5.0% | 6.0% |

### 护栏指标（防止伤害）
| 指标 | 底线 | 超限动作 |
|------|------|---------|
| 人均GMV | 不低于-5% | 自动停止实验 |
| 退货率 | 不高于+3% | 告警 + 人工介入 |

### 观察指标（了解原因）
- 页面停留时长
- 跳出率
- 人均浏览商品数
```

### Phase 4: 实验执行

```markdown
## 实验流程

### 第1天
- 小流量验证（5%用户，各2.5%）
- 检查数据上报是否正常
- 观察有无明显异常（崩溃、投诉）

### 第2-5天
- 全量开实验（各50%）
- 每日监控核心指标和护栏指标
- 不中途查看显著性（避免peeking问题）

### 第6天
- 计算最终统计显著性
- 输出结论报告
```

**禁忌：**
- ❌ 频繁查看结果，看到显著就停
- ❌ 样本不够就下结论
- ❌ 多组对比不校正显著性（Bonferroni校正）

### Phase 5: 结果分析

```markdown
## 实验结果

### 数据汇总
| 指标 | A组（对照） | B组（实验） | 差异 | 置信区间 | 显著性 |
|------|-----------|-----------|------|---------|--------|
| 转化率 | 5.02% | 5.89% | +0.87pp | [0.45, 1.29] | p<0.05 ✅ |
| 人均GMV | ¥128 | ¥131 | +2.3% | [-1.2%, 5.8%] | 不显著 |
| 退货率 | 8.1% | 8.3% | +0.2pp | [-0.8, 1.2] | 不显著 |

### 统计显著性
- 检验方法：双样本Z检验
- p值：0.003（<0.05，拒绝H0）
- 功效：82%
- 结论：实验组转化率显著高于对照组

### 业务结论
✅ **实验成功**，建议全量上线B方案
- 转化率提升17.3%（相对提升）
- 无护栏指标显著恶化
- 预计年增收：¥XXX万
```
