# Cohort Analysis - Cohort分析

---

## Description

通过同期群分析，看清用户留存的真实趋势。

**解决的实际问题：**
- 留存数字好看，但不知道是新老用户混杂
- 不知道哪批用户质量好
- 产品改进后，留存真的提升了吗？

---

## When to Use

**强制触发条件（任一满足）：**
- 明确说"cohort分析"、"同期群"
- 留存数字波动大，需要下钻
- 对比不同渠道/版本的用户质量
- 验证产品改进效果

**不适用场景：**
- 产品刚上线（<30天数据）
- 用户量<1000（样本太小）

---

## Pre-conditions

**必须提供：**
- 用户注册时间数据
- 活跃行为定义（登录/使用核心功能）
- 时间范围（建议≥30天）

---

## Workflow

### Phase 1: 定义Cohort

```markdown
## Cohort定义方式

### 按注册日期（最常用）
- Cohort = 同一周/月注册的用户
- 看这批用户后续留存

### 按渠道
- Cohort = 同一渠道来的用户
- 对比渠道质量

### 按行为
- Cohort = 完成某行为的用户
- 如：首次付费用户

### 本次分析定义
- Cohort维度：注册周
- 观察指标：7日留存
- 时间范围：过去12周
```

### Phase 2: 制作Cohort表

```markdown
## Cohort表结构

| Cohort | 用户数 | W0 | W1 | W2 | W3 | W4 |
|--------|--------|----|----|----|----|----|
| 2026-W01 | 100 | 100% | 45% | 38% | 35% | 33% |
| 2026-W02 | 120 | 100% | 42% | 36% | 32% | - |
| 2026-W03 | 110 | 100% | 48% | 40% | - | - |

### 解读
- 行：同一批用户（同期群）
- 列：时间推移（周0=注册周）
- 对角线：同一时间点不同Cohort的对比
```

### Phase 3: 数据分析

```markdown
## 分析维度

### 1. 纵向看（同一Cohort趋势）
```
W01: 100% → 45% → 38% → 35% → 33%
      ↓     ↓     ↓     ↓
    首周流失55%，之后稳定
```
**结论**：流失主要发生在第一周，需优化新用户体验

### 2. 横向看（同期对比）
```
W1留存：W01=45%, W02=42%, W03=48%
```
**结论**：W03 cohort质量更好，可能渠道优化有效

### 3. 颜色热力图
```
绿色 > 45% | 黄色 35-45% | 红色 < 35%
```
**快速识别**：哪段时间 cohort 质量好/差

### 4. 计算平均留存曲线
```
Week 0: 100%
Week 1: avg(45%, 42%, 48%, ...) = 45%
Week 2: avg(38%, 36%, 40%, ...) = 38%
```
```

### Phase 4: 洞察与行动

```markdown
## 常见Cohort模式

### 模式1：正常留存
```
100% → 50% → 40% → 35% → 33% → 32%
         ↓
    快速下降后稳定
```
**健康度**：✅ 正常

### 模式2：持续下降（危险）
```
100% → 50% → 35% → 25% → 18% → 12%
         ↓
    没有稳定趋势
```
**问题**：产品核心价值未被感知
**行动**：用户调研找原因

### 模式3：后期回升（良好）
```
100% → 45% → 38% → 40% → 42%
                  ↗
    第三周开始回升
```
**可能原因**：习惯养成/社区效应
**行动**：强化中期激励

### 模式4：新Cohort质量下降
```
W01: 45%, W02: 42%, W03: 35%, W04: 30%
                      ↓
                逐渐变差
```
**可能原因**：渠道质量下降/竞争加剧
**行动**：检查渠道投放/竞品动态
```

## Output

```markdown
# Cohort分析报告

## 数据概览
[热力图表格]

## 关键发现
1. 首周流失率55%，是主要流失点
2. W03 cohort质量提升（渠道优化见效）
3. 长期留存稳定在32%左右

## 对比分析
| Cohort | W1留存 | W4留存 | 质量 |
|--------|--------|--------|------|
| 自然流量 | 50% | 38% | ⭐⭐⭐ |
| 付费广告 | 40% | 28% | ⭐⭐ |

## 建议
1. 优化首周体验（目标W1留存>50%）
2. 加大自然流量投入
3. 付费广告优化素材，提升精准度
```
